My Projects

I'm most proud of ...

Gameify Github

An object-oriented 2d game engine developed from scratch in vanilla javascript. Gameify runs in the browser, offering a simple and accesible way to make 2d games.

WeatherForMe Visit

A weather app for anywhere in the U.S. WeatherForMe uses data from the National Weather Service and presents it as nice graphs. Created with Node.js and Chart.js.

SimpleFile Github

A minimalist web-based file browser for easily downloading files over wifi. It's designed to be quick to set up, to make simple transfers easy. I originally created it in a pinch because my phone and computer refused to play nice, and have refined it over time.

Solvespace Github

I designed new icons for Solvespace (an open-source 3d CAD program). I recreated nearly all of the icons in the program, and they were shipped with it in release 3.0.
