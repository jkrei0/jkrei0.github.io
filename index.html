My Portfolio

I'm most proud of ...

WeatherFor.Me Live Website

A weather app for anywhere in the US that takes data from the National Weather Service and presents it as nice graphs, created with Node.js and Chart.js. View it at weatherfor.me.

Gameify Source Code

An object-oriented 2d game engine developed from scratch in vanilla javascript. Gameify runs (and exports games) in the browser, offering a simple and accesible way to make 2d games.

Solvespace Website

I helped design new icons for solvespace (a 3d CAD modeler). I redesigned nearly all of the icons in the program, and they were shipped with it in release 3.0.

Graphing Calculator Source

A graphing calculator created using c++ that runs in the terminal. It includes a basic lexer and solver, support for many common functions (eg sine and cosine), and can graph any implicit equation.

Some smaller projects

SimpleFile Source Code

A minimalist web-based file server made with Node.js, useful for transferring / exploring files over WiFi. Designed to be minimalist and easy to set up in order to make simple transfers easy. I originally created it to easily tranfer app icons I designed for my phone, and refined it over time.

Pyroide Source Code

A web-based code editor built upon the Ace code editor. It offers workspaces in which many people can collaborate on code at once. I also built a desktop version which offered more functionality. I am no longer developing pyroide and have no plans to release the desktop version.

Beacon Source Code

(Was) a fork of the Caret editor for Chrome. It incorporated the Beagle Term serial terminal emulator to allow easy programming of Arduinos, NeoTrellises, and other devices from Chrome/ChromeOS.

Window saver Source Code

A python script to save your window arrangements. The script uses pywin32 to save & restore the positions of your windows. Useful for quickly swapping workflows, or when frequently disconnecting/reconnecting an external monitor.
