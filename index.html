My Portfolio

This is a collection of projects I've created during my programming journey.

Web Development

WeatherFor.Me View project

A simple weather app for anywhere in the US that takes data from the National Weather Service and presents it as nice graphs, created with Node.js and Chart.js. View it at weatherfor.me.

Pyroide See on Github

Pyroide is a web-based code editor built upon the Ace code editor. It offers workspaces in which many people can collaborate on code at once. I also built a desktop version which offered more functionality. I am no longer developing pyroide and have no plans to release the desktop version.

Beacon See on Github

Beacon was a fork of the Caret code editor for Chrome. It incorporated the Beagle Term serial terminal emulator to allow easy programming of Arduinos, NeoTrellises, and other devices from Chrome/ChromeOS.

Onu! See on Github

An Uno clone built to learn Vue.js The backend is built with node.js and socket.io (mostly because that was the easy way - this project was about Vue, not backends)

Miscellaneous Projects

Solvespace See website

I helped design new icons for solvespace (a 3d CAD modeler). I designed improved icons for the toolbar and property broswer, which were integrated in release 3.0.

Graphing Calculator View

A graphing calculator created using c++ that runs in the terminal. It includes a basic lexer and solver, support for many common functions (eg sine and cosine), and can graph any implicit equation.

Window saver Download

A python script to save your window arrangements. The script uses pywin32 to save & restore the positions of your windows. Useful for quickly swapping workflows, or when frequently disconnecting/reconnecting an external monitor.
